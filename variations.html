The Claude Code Gazette

Volume III · April 15, 2026 · Daily Edition
Breaking · April 13
Claude Code v2.1.105 Ships PreCompact Hooks, Enabling Plugins to Block Context Compaction
The latest release introduces a PreCompact lifecycle hook that gives plugins veto power over context window compaction. By returning exit code 2 or a JSON {"decision":"block"} response, hooks can prevent the system from discarding context prematurely. This is paired with background monitor support via a new monitors manifest key, auto-arming at session start. The EnterWorktree tool now accepts a path parameter for switching into existing worktrees. Additionally, /proactive becomes an alias for /loop, and stalled API streams abort after 5 minutes with automatic non-streaming retry.
Security · April 10
CVE-2026-35021: Command Injection in POSIX which Fallback Patched
v2.1.101 fixes a command injection vulnerability in the POSIX which fallback path. Also ships OS CA certificate store trust by default, eliminating enterprise TLS proxy setup friction.
